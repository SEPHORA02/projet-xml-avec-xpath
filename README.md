# projet-xml-avec-xpath
# Catalogue de Films en XML

# Description
Ce projet est un catalogue de films développé pour maîtriser la technologie XML et l'utilisation des filtres XPath. Il permet de stocker des informations sur des films (titre, année, réalisateur, genre, note, etc.) et de rechercher des films selon différents critères.

# Objectifs
- Apprendre à structurer des données en XML.
- Utiliser XPath pour filtrer et rechercher dans un fichier XML.
- Générer des listes de films selon différents critères (genre, année, réalisateur, note IMDb…).
- Préparer des transformations XML vers HTML/XSLT pour une présentation web.

# Fonctionnalités
- Catalogue hiérarchisé avec plusieurs niveaux (par genre, par décennie, etc.)
- Recherche de films via XPath selon plusieurs critères.
- Possibilité d’ajouter dynamiquement des attributs (ex : note IMDb).
- Transformation XML → HTML pour affichage web via XSLT.
- Version web disponible pour consultation des films.


# Technologies utilisées
- XML (Extensible Markup Language)
- XPath (filtres et recherches)
- XSLT (pour transformation HTML)
- php; html, js , css
